# Retail-sales-analysis-superstore
Análisis de rentabilidad y desempeño comercial usando Excel, SQL Server y Power BI

🛠 Herramientas utilizadas:
- Excel (limpieza y transformación)
- SQL Server (análisis)
- Power BI (visualización)

📂 Dataset
-Sample Superstore Dataset (Kaggle)

🔍 Análisis realizado
- Evaluación de ventas y ganancias
- Identificación de productos con pérdidas
- Impacto de descuentos en la rentabilidad
- Análisis por región
- Tendencia de ventas en el tiempo
